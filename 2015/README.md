# Advent of Code 2015

Solutions for [Advent of Code 2015](https://adventofcode.com/2015)

## Days Completed

- [x] Day 1: Not Quite Lisp

## Running Solutions

To run a specific day's solution:

```bash
cd 2015
python day01.py
```

Make sure you have your puzzle input saved as `input.txt` in the same directory, or the solution will use example data.
